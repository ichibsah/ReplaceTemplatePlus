
This plugins allows the user to:

Replace template while importing in element properties


Highlights:
 1. Uses Jquery and AJAX to send RQL
 2. Easy to install
 3. Other than RQL communication, all processing are done client side.  Minimal server load.

  
Installation:
 1.  Unzip zip (including folder within) to Management Server plugins directory
 2.  Import plugin via Server Manager interface
 3.  Assign plugin to project
 

Accessibility:
In SmartTree, select template, "Replace Template Plus" from action menu
